# 📌 Queue Using Dynamic Array (OOP)

## 📖 Overview

This project — **Queue Using Dynamic Array (OOP)** — implements a Queue data structure in C++ using a custom-built Dynamic Array class.

Instead of relying on STL containers, the Queue is built manually to demonstrate how data structures operate internally and how memory is managed dynamically.

The project highlights how a higher-level data structure (Queue) can be constructed on top of a lower-level structure (Dynamic Array).

---

## 🎯 Learning Purpose

This project is designed to strengthen understanding of Data Structures and OOP concepts.

Through this implementation, we learn how to:

* Build Queue from scratch using a Dynamic Array
* Understand FIFO (First In, First Out) behavior
* Manage data using manual memory structures
* Apply Object-Oriented Programming principles
* Reuse existing data structures to build new ones

---

## ⚙️ Core Functionality

### 🔹 Queue Operations

* Push (enqueue) → insert element at the end
* Pop (dequeue) → remove element from the front
* Front → access first element
* Back → access last element
* Size → get number of elements
* IsEmpty → check if queue is empty

---

### 🔹 Extended Features

* Reverse queue
* Update element by index
* Insert after specific index
* Insert at front or back
* Clear entire queue

---

## 🧠 Educational Insight

This project demonstrates an important concept:

**“Data structures can be built on top of each other.”**

By implementing a Queue using a Dynamic Array, we understand how abstraction allows us to reuse logic and build more complex systems efficiently.

---

## ⚠️ Performance Note

* Push operation → efficient
* Pop operation → less efficient (requires shifting elements)

This highlights a limitation of array-based queues and motivates more advanced implementations like Circular Queue.

---

## 🧩 Project Philosophy

The design follows key OOP principles:

* **Encapsulation** → Queue logic is separated from array implementation
* **Abstraction** → User interacts with simple queue operations
* **Code Reusability** → Dynamic Array is reused as a base structure
