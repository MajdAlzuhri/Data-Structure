# 📌 Double Linked List - Data Structures Project

## 📖 Overview

This project implements a **generic Doubly Linked List** using C++ templates.
It demonstrates core Data Structures concepts along with Object-Oriented Programming (OOP) principles such as encapsulation and modular design.

---

## 🚀 Features

* Insert at beginning
* Insert at end
* Insert at specific index
* Delete node (first, last, specific)
* Search (Find)
* Reverse the list
* Update node value
* Get node by index

---

## 🛠️ Technologies Used

* C++
* Templates (Generic Programming)
* Object-Oriented Programming (OOP)

---

## 🧠 Concepts Applied

* Doubly Linked List
* Dynamic Memory Allocation
* Pointers (Next & Previous)
* Encapsulation
* Modular Design

---

---

## 💡 Example Usage

```cpp
clsDblLinkedList<int> list;

list.InsertAtBeginning(10);
list.InsertAtEnd(20);
list.InsertAtEnd(30);

list.PrintList(); // Output: 10 20 30
```

---

## 📌 Notes

* The implementation uses templates to support different data types.
* Memory management is handled manually using pointers.
